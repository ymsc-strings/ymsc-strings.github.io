研究简介

苏宁的研究以 bootstrap 为核心工具，目标是在不依赖拉氏量微扰展开的情况下约束和求解强耦合量子场论。近五年，他的工作覆盖 3d Ising 模型和超 Ising 模型的高精度数据、navigator function 等数值算法、非阿贝尔流 bootstrap、解析 lightcone bootstrap 与数值 bootstrap 的混合方法，以及用 S-matrix bootstrap 拟合 pion scattering 数据。这些工作既推进了共形场论的数值精度，也把 bootstrap 方法扩展到散射振幅和强相互作用现象。
